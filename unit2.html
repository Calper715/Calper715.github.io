Welcome to the Unit2 page. Here you can see the work related to Unit2.

  • These are the concepts I have learned over Unit2.
  • Object-oriented programming: A programming paradigm based on the concept of "objects".
  • Classes: Blueprints for creating objects.
  • Inheritance: A mechanism for creating a new class using properties and methods of an existing class.
  • Encapsulation: The bundling of data and methods that operate on that data.
  • Polymorphism: The ability to present the same interface for different data types.
  • Data structures: Ways to organize and store data for efficient access and modification.
  • APIs: Interfaces that allow different software applications to communicate with each other.
  • Debugging: The process of identifying and removing errors from code.
